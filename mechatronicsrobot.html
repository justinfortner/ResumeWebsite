About the MEchatronics Final Robot

This project, as shown in the video link above, combines arduino hardware, custom built sensors, C programming and laser cut materials to produce a fully autonomous robot to deposit a ping pong ball inside of a correct hole within a beacon tower. A state machine coded in C software allowed the arduino hardware to communicate with the many custom build sensors. This robot was able to use an IR signal sensor to lock onto desired “towers”, bumper sensors to signal arrival at the tower and easy circumnavigation, tank circuits to read the correct wire signal emitted by the wall and a tape sensor to locate the correct hole in which to deposit a ping pong ball. This project was the collaborative effort of myself and two other classmates. We were able to complete this project quickly and effectively through hard work and communication.
